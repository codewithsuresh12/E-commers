## 📌 Project Overview

This E-Commerce Website is a modern online shopping platform designed to provide users with a seamless shopping experience. It includes product browsing, cart management, secure authentication, and responsive design for all devices.

---

## ✨ Features

### 👤 User Features

* User Registration & Login
* Product Search & Filtering
* Product Categories
* Add to Cart
* Wishlist Management
* Order Placement
* User Profile Management
* Responsive Design

### 🛠️ Admin Features

* Product Management
* Category Management
* Order Management
* User Management
* Dashboard Analytics

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/ecommerce-website.git
```

### Move to Project Folder

```bash
cd ecommerce-website
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```
